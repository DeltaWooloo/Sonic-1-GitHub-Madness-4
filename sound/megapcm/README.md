# Mega PCM 2 Release bundles

Source code and documentation: https://github.com/vladikcomper/MegaPCM/tree/2.x

Mega PCM 2 is released in a form of single easy to include .asm file and supports several M68K assemblers. Directories you see represent different assembler targets. Pick the right directory according to the installation guide for your disassembly.

- `as`: For the AS Macro Assembler;
- `asm68k`: For the ASM68K assember;
- `asm68k-linkable`: For ASM68K + PsyLink build systems (you probably won't need this one).
